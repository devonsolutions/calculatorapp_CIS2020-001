About

This is a calculator web app. The web app's functionality is built off of HTML, CSS, and JavaScript. The "How to Use" and "About" pages consist of HTML and CSS. The Flask web framework builds the structure and connectivity of the app's components.

The repository consists of a multitude of components. The "static" folder contains the CSS file. The "templates" folder holds base HTML file and the additional, extending HTML files. The "app.py" file connects the HTML files. The "requirements.txt" file lists the packages required for the program to run. Corrections to the code were made by DeepSeek AI.
